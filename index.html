Pirate Donation Buttons

Never heard of Pirate before? Well, let me be the first to share the good news about this excellent cryptocurrency. Pirate (symbol: ARRR) a relatively new coin, kicks the collective asses off of other major Privacy coins. If you like logic more than hype, here is a reddit post that explains the rationale in pretty clear terms.

But how does one actually go about getting some ARRR? Well don't waste time sailing the digital seas searching for Pirate ARRR when you can simply place a friendly donate button on yarr webpage!

Find the JavaScript booty and instructions for use on github at: https://github.com/dogracer/donate_arrr

Simple to use: just wrap your favorite HTML element to make it into a donation flyout on hover (mouse) or tap (touch device):

<a href="piratechain:YOUR-zADDRESS">☠️ DONATE ARRR</a>

Here's how it looks like this in practice:

☠️ DONATE ARRR

You can wrap images too if you have some pirate arrrt you wish to display.

Enjoy!
